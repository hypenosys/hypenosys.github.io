CONFIGURACIÓN API DE IA
Tu API key se guarda solo en este navegador. Nunca se sube a GitHub.
¿Ollama no conecta?

Asegúrate de que Ollama está arrancado con OLLAMA_ORIGINS=*.

En macOS/Linux:

OLLAMA_ORIGINS=* ollama serve

En Windows: Establece la variable de entorno del sistema OLLAMA_ORIGINS a * y reinicia Ollama.

Seguridad y CORS (NVIDIA NIM)

Las llamadas directas desde el navegador a integrate.api.nvidia.com pueden fallar por políticas de CORS y exponen tu API Key en las cabeceras.

Recomendación: Utiliza un proxy seguro (Cloudflare Worker o similar) que inyecte la clave en el lado del servidor.

Si usas NIM local/self-hosted (HTTP), asegúrate de permitir el origen de este sitio o usa una extensión de CORS.
